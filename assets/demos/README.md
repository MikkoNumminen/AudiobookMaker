# Demo audio clips

Short samples generated by AudiobookMaker for the README "Hear it first"
section and for the Make Sample feature's built-in fallback. All source
texts are public domain.

| File | Language | Engine | Voice | Source |
|------|----------|--------|-------|--------|
| `finnish_grandmom_kivi.mp3` | Finnish | Chatterbox (T3 finetune) | Grandmom | Aleksis Kivi, *Seitsemän veljestä* (1870, public domain) |
| `english_grandmom_gibbon.mp3` | English | Chatterbox (multilingual + voice clone) | Grandmom | Edward Gibbon, *The Decline and Fall of the Roman Empire* (1776, public domain) |

Generated 2026-04-16 on RTX 3080 Ti at ~1.2x realtime.

## Regenerating

From the repo root with the Chatterbox venv installed:

```bash
# Finnish
.venv-chatterbox/Scripts/python.exe scripts/generate_chatterbox_audiobook.py \
  --text-file <your_finnish_input.txt> --out assets/demos --language fi --device cuda

# English
.venv-chatterbox/Scripts/python.exe scripts/generate_chatterbox_audiobook.py \
  --text-file <your_english_input.txt> --out assets/demos --language en --device cuda
```

Then rename the output `01_Text.mp3` to the descriptive filename.
